# RbmAndroidClient
Android client for the RBM backend

Add to gradle

```gradle
repositories {
    maven {
        url 'https://github.com/flogvit/RbmAndroidClient/raw/master/release/'
    }
}

dependencies {
//    ..
    compile 'com.github.nkzawa:engine.io-client:0.5.0'
    compile 'com.cellarlabs:rbmandroidclient:0.1.0@aar'
}
```
